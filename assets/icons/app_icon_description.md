# DrinkTime App Icons

This directory should contain the app launcher icons for different platforms.

## Icon Design
The main app icon follows the "DT" logo design from the concept images:
- Blue rounded square background (color: #87CEEB - Sky Blue)
- White "DT" text in Fredoka font, bold
- Small orange drink glass icon in bottom-right corner
- Drop shadow for depth
